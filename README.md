# Pytorch_beginner

This is a beginner tutorial for Pytorch with well-structured code.

## Prepare Data
Create d directory under root in such hierarchical structure:

```
/data/
     /train/
          /0/
          /1/
          /2/
          ...
    /test/
          /0/
          /1/
          /2/
          ...                 
```

## Train (and test)
You can change or add configurations inside options.py or config.py

Run command:

```
python train.py
```
